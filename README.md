# Internet of Things Application Development
# Description
Develop a new, complete project with functionalities that differ by at least 30% from the [original code base](https://github.com/nhanksd85/YoloUNO_PlatformIO/tree/RTOS_Project)

# Requirements
Using PlatfromIO (Arduino frameworks) and YoloUNO (ESP32-S3) to completed these tasks:
* Task 1: Single LED Blink with Temperature Conditions
* Task 2: NeoPixel LED Control Based on Humidity
* Task 3: Temperature and Humidity Monitoring with LCD Display (Remove ALL global variables in your project by using semaphore in RTOS)
* Task 4: Web Server in Access Point Mode
* Task 5: TinyML Deployment & Accuracy Evaluation
* Task 6: Data Publishing to CoreIOT Cloud Server
